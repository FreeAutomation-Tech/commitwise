# Contributing

Thanks for wanting to make CommitWise better!

## Development

```bash
git clone https://github.com/FreeAutomation-Tech/commitwise.git
cd commitwise
pip install -e ".[dev]"
```

## Code Style

- Type hints everywhere
- Follow PEP 8
- 100 char line limit

## PR Process

1. Fork the repo
2. Create a feature branch
3. Write/update tests
4. Run `pytest`
5. Open a PR
